# 🍳 Кулинарная книга

Десктопное приложение на **Python + CustomTkinter + SQLite**.

## Структура проекта

```
cookbook/
├── main.py        # Главный файл — UI, логика
├── database.py    # Модуль базы данных (SQLite)
├── recipes.db     # Создаётся автоматически при первом запуске
└── README.md
```

## Установка и запуск

```bash
pip install customtkinter pyperclip
python main.py
```

## Возможности

- **3 вкладки**: Добавить рецепт / Все рецепты / Избранное
- **CRUD**: создание, редактирование, удаление рецептов
- **Поиск**: по ингредиенту (регистронезависимый) и по категории
- **Избранное**: отмечать при создании, переключать в просмотре
- **Просмотр**: двойной клик — полное окно рецепта (Toplevel)
- **Экспорт в TXT**: красиво оформленный текстовый файл
- **Копирование ингредиентов** в буфер обмена

## Горячие клавиши

| Клавиша | Действие                         |
|---------|----------------------------------|
| Ctrl+N  | Перейти к форме нового рецепта   |
| Ctrl+F  | Переключить избранное у выбранного |
